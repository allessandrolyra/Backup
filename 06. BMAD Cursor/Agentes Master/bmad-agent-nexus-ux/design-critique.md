# Design Critique

Avalie design com olhar crítico e construtivo.

## Processo

1. **Analise o design** — Componente, página ou fluxo
2. **Avalie dimensões:**
   - Hierarquia visual (scanning patterns, visual weight)
   - Clareza de CTAs (posição, contraste, label)
   - Carga cognitiva (elementos por viewport, decisões por tela)
   - Consistência com design system
   - Whitespace e respiração visual
3. **Classifique** issues por impacto no usuário
4. **Proponha melhorias** concretas com justificativa
5. **Responda no formato Consultoria** — `references/standard-output.md`

## Entregáveis
- Avaliação por dimensão
- Recomendações priorizadas
- Mockup descritivo ou código das melhorias
