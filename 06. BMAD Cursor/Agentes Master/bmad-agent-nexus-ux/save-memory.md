# Salvar no Project Memory

Salve decisões, componentes e padrões na memória persistente.

## Processo

1. **Carregue** `references/project-memory.md` para estrutura
2. **Leia o Project Memory atual** em `{project-root}/_bmad/memory/bmad-agent-nexus-ux-ledger/index.md`
3. **Se não existe**, crie seguindo a estrutura completa (seções A-F)
4. **Se existe**, identifique o que mudou
5. **Mostre o diff** — Antes → Depois
6. **Peça confirmação** antes de salvar

## Regras
- Nunca sobrescrever decisões — adicionar como nova entrada com data
- Componentes criados devem ter: nome, path, propósito
