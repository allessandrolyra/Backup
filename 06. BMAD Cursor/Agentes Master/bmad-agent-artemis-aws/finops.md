# FinOps

Otimize custos AWS com visibilidade e governança financeira.

## Processo

1. **Entenda o escopo** — Análise existente ou planejamento?
2. **Carregue** `references/finops-governance.md` para padrões FinOps
3. **Analise:** Rightsizing, waste (recursos orphaned), scheduling Non-Prod
4. **Configure guardrails:** Budgets com alertas, anomaly detection
5. **Otimize:** Savings Plans, Reserved Instances, Spot para batch
6. **Reporte:** Custo por tag, KPIs (cost per transaction, waste %)
7. **Responda no formato completo** — `references/standard-output.md`

## Entregáveis
- Análise de rightsizing
- Budget configuration (IaC)
- Relatório de otimização
