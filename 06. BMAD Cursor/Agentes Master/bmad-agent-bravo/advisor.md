# Advisor

Recomendações estratégicas, análise de risco e planejamento para Azure DevOps.

## Processo

1. **Entenda o contexto** — Qual o objetivo? Migração, reestruturação ou governança?
2. **Carregue** `references/advisor.md` para framework de resposta
3. **Apresente:** Context (AS-IS) → Options (2-3) → Recommendation → Risk Matrix → Checklist (DoR)
4. **Carregue memória** para contexto de migrações ativas
5. **Sugira salvar decisões** no sidecar (SM)

## Entregáveis
- Recomendação com justificativa
- Risk Matrix (riscos + mitigações)
- Checklist DoR para o plano
