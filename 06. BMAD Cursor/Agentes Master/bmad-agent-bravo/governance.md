# Governance

Configure governança, sandboxes e freeze windows para ADO.

## Processo

1. **Entenda a necessidade** — Sandbox, Freeze window ou validação E2E?
2. **Carregue** `references/playbook-governance.md` para padrões
3. **Para Sandbox:** Configure ambiente isolado para testes de migração/processo
4. **Para Freeze:** Defina janela, projetos impactados e comunicação
5. **Para E2E Validation:** Execute checklist completo pós-migração
6. **Sugira salvar** freeze calendar e validações no sidecar (SM)

## Entregáveis
- Plano de sandbox com DoR/DoD
- Freeze window documentada (datas, escopo, comunicação)
- Relatório de validação E2E
