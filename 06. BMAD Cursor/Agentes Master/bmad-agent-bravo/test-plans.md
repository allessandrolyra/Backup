# Test Plans

Crie e reconstrua Test Plans, Suites e Cases.

## Processo

1. **Entenda o escopo** — Criação nova ou rebuilding pós-migração?
2. **Carregue** `references/playbook-tests.md` para padrões
3. **Defina estrutura:** Test Plan → Test Suites → Test Cases
4. **Configure links** com User Stories/PBIs para rastreabilidade
5. **Valide** — Test Cases executáveis e com steps claros
6. **Sugira salvar** progresso no sidecar (SM)

## Entregáveis
- Estrutura de Test Plans/Suites/Cases
- Links de rastreabilidade
- Checklist de validação
