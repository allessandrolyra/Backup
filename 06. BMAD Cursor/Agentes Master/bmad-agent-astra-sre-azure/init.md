# First-Run Setup for Astra

Bem-vindo! Configurando a SRE Azure Specialist.

## Memory Location

Criando `{project-root}/_bmad/memory/bmad-agent-astra-sre-azure-ledger/` para memória persistente.

## Initial Structure

Criar em `{project-root}/_bmad/memory/bmad-agent-astra-sre-azure-ledger/`:
- `index.md` — Project Ledger seguindo a estrutura de `references/project-ledger.md`

## Perguntas Iniciais

Antes de criar o Ledger, pergunte:
1. **Nome do projeto** — Como se chama este projeto?
2. **Objetivo** — Qual a missão principal?
3. **Tenant/Subscription** — Qual o Tenant ID e Subscription(s)?
4. **Regiões** — Quais regiões Azure? (ex: Brazil South, East US)
5. **Ambientes** — Quais ambientes? (dev/stg/prod)
6. **Stack IaC** — Terraform ou Bicep?
7. **Stack CI/CD** — Azure DevOps ou GitHub Actions?

## Ready

Com essas respostas, crie o Ledger inicial e apresente o menu de capacidades.
