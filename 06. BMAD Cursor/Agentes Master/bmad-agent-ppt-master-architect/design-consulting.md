# Design Consulting

Provide expert guidance on presentation design: layout, typography, branding, visual hierarchy, color systems, accessibility, and visual composition.

## Activation

When the user needs design guidance, assess their need and route accordingly:

### Layout & Composition

**Slide Layout Patterns:**

| Content Type | Recommended Layout |
|-------------|-------------------|
| Title / Section Divider | Centered text, bold typography, minimal elements |
| Key Message | Large statement, supporting subtitle, background image |
| Bullet Points | Left-aligned text, right visual/image, 3-5 bullets max |
| Comparison | 2 or 3 columns, parallel structure, consistent formatting |
| Data / Chart | Chart occupies 60-70% of slide, insight headline above |
| Timeline | Horizontal or vertical flow, milestones marked, color-coded |
| Quote / Testimonial | Large quote, attribution below, subtle background |
| Team / People | Grid of photos with name/role, consistent sizing |
| Process / Flow | Numbered steps or flowchart, visual connectors |
| Summary / Takeaways | 3-5 key points, numbered or icon-paired |

**Composition rules:**
- Maintain consistent margins across all slides (minimum 0.5 inch / 1.27cm)
- Use alignment guides — never eyeball positioning
- Apply the rule of thirds for image placement
- White space is a design element, not wasted space
- One focal point per slide

### Typography

**Font Pairing Recommendations:**

| Context | Title Font | Body Font |
|---------|-----------|-----------|
| Corporate / Enterprise | Segoe UI Bold | Segoe UI Regular |
| Modern / Startup | Montserrat Bold | Open Sans Regular |
| Classic / Formal | Georgia Bold | Calibri Regular |
| Technical | Roboto Bold | Roboto Regular |
| Microsoft Standard | Aptos Display | Aptos |

**Size guidelines for 16:9 presentations:**

| Element | Minimum Size | Recommended Size |
|---------|-------------|-----------------|
| Slide Title | 28pt | 32-36pt |
| Subtitle | 20pt | 22-24pt |
| Body Text | 16pt | 18-20pt |
| Bullet Points | 16pt | 18pt |
| Footnotes / Source | 10pt | 12pt |
| Chart Labels | 10pt | 12-14pt |

**Typography rules:**
- Maximum 2 font families per presentation
- Consistent weight hierarchy: Bold for titles, Regular for body, Light sparingly
- Avoid decorative fonts in corporate contexts
- Line spacing: 1.2 to 1.5 for body text
- Never use all caps for more than 3-4 words

### Color System

**Building a presentation color palette:**

1. **Primary color** — Main brand color or dominant deck color (headers, key accents)
2. **Secondary color** — Complementary color for variety (subheadings, secondary elements)
3. **Neutral base** — Background and text colors (white/light gray background, dark gray/black text)
4. **Accent color** — For highlights, CTAs, emphasis (use sparingly)
5. **Chart colors** — 4-6 distinct colors for data visualization

**Contrast requirements:**
- Text on background: minimum 4.5:1 contrast ratio for body text
- Large text (24pt+): minimum 3:1 contrast ratio
- Chart elements: distinguishable without relying solely on color
- Test readability from 3 meters / 10 feet distance

### Visual Hierarchy

**Hierarchy layers (from most to least prominent):**

1. Slide title — Largest, boldest, top of slide
2. Key message / data point — Second largest, central positioning
3. Supporting content — Standard body size, aligned left
4. Supplementary info — Smaller, lighter color, peripheral position
5. Source / footnotes — Smallest, bottom of slide

**Visual weight tools:**
- Size (larger = more important)
- Color (saturated = more important)
- Position (top-left = seen first in Western reading patterns)
- Contrast (high contrast = draws attention)
- Isolation (surrounded by white space = emphasized)

### Branding Application

**When corporate brand guidelines exist:**

1. Extract primary and secondary colors from brand guide
2. Map brand fonts to presentation hierarchy
3. Place logo consistently (typically bottom-right or top-right, per guidelines)
4. Use brand-approved icon style
5. Follow template layouts when available
6. Maintain brand photography style (if applicable)

**When no brand guidelines exist:**

1. Propose a neutral professional palette
2. Use universally professional fonts (Segoe UI, Calibri, Aptos)
3. Suggest a clean, minimal layout system
4. Offer 2-3 palette options for user selection

### Accessibility

**Presentation accessibility checklist:**

- [ ] Sufficient color contrast (4.5:1 body, 3:1 large text)
- [ ] Information not conveyed by color alone (use shapes, patterns, labels)
- [ ] Alt text for all images and charts
- [ ] Readable font sizes (minimum 16pt body, 28pt titles)
- [ ] Logical reading order set in Selection Pane
- [ ] Unique, descriptive slide titles
- [ ] No auto-advancing animations that create barriers
- [ ] Table headers properly marked
- [ ] Links have descriptive text (not "click here")
- [ ] Run PowerPoint's Accessibility Checker before finalizing
