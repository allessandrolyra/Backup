# Storytelling & Narrative Structure

Structure presentations with logical narrative flow that engages the audience, communicates clearly, and drives desired outcomes.

## Activation

When the user needs help with presentation narrative, storyline, or sequencing, follow this process:

### Step 1 — Define the Narrative Context

Identify:

1. **What is the desired outcome?** — Decision, approval, awareness, action, buy-in
2. **Who is the audience?** — Their priorities, concerns, and decision-making style
3. **What is the core message?** — The one thing the audience must remember
4. **What is the emotional arc?** — From current state to desired state

### Step 2 — Select Narrative Framework

Choose the framework that best fits the context:

**Executive Decision Framework**
```
1. Context — Why are we here?
2. Key Finding/Insight — What did we discover?
3. Implications — What does this mean?
4. Options — What are the choices?
5. Recommendation — What should we do?
6. Next Steps — What happens now?
```

**Problem-Solution Framework**
```
1. Current State — Where are we?
2. Problem/Challenge — What's wrong?
3. Impact — Why does it matter?
4. Proposed Solution — What do we recommend?
5. Benefits — What do we gain?
6. Implementation — How do we do it?
7. Timeline & Investment — When and how much?
```

**Results & Progress Framework**
```
1. Period Overview — What period are we covering?
2. Key Achievements — What went well?
3. Metrics & KPIs — How did we perform?
4. Challenges — What didn't go as planned?
5. Lessons Learned — What did we learn?
6. Outlook & Next Steps — What's ahead?
```

**Persuasion Framework (Pitch/Proposal)**
```
1. Hook — Attention-grabbing opening
2. Pain Point — The problem worth solving
3. Vision — The world after the solution
4. Solution — How we get there
5. Proof — Evidence it works (data, cases, testimonials)
6. Call to Action — What we're asking for
```

**Strategy Framework**
```
1. Strategic Context — Market, competitive, organizational
2. Vision & Objectives — Where we want to be
3. Analysis — SWOT, gaps, opportunities
4. Strategic Pillars — Key initiatives
5. Roadmap — Phased approach
6. Risks & Mitigation — What could go wrong
7. Success Metrics — How we'll measure progress
```

**Training/Workshop Framework**
```
1. Learning Objectives — What participants will gain
2. Agenda — Session structure
3. Core Content — Modular teaching blocks
4. Interactive Elements — Exercises, discussions, Q&A
5. Summary — Key takeaways
6. Resources — Further reading, tools, contacts
```

### Step 3 — Build the Storyline

For each narrative block, define:

```
Block: [Name]
Slides: [Estimated count]
Purpose: [Why this section exists]
Key Message: [What the audience takes away]
Transition to Next: [How it connects to the next block]
```

Present the complete storyline and refine with user input.

### Step 4 — Opening & Closing Strategy

**Strong Openings:**
- Start with a compelling data point or statistic
- Open with a question that resonates with the audience
- Present a before/after contrast
- Share a relevant brief anecdote or scenario
- State the most important conclusion first (executive audiences)

**Strong Closings:**
- Restate the core message with emphasis
- Clear call-to-action or decision request
- Forward-looking statement connecting to audience priorities
- Summary slide with 3 key takeaways maximum
- End with the "so what" — why this matters

### Step 5 — Transition Design

Between sections, ensure:

- Visual or textual transition slides for major section changes
- Each section connects logically to the next
- The "why" of the next section is clear from the current one
- Progression feels natural, not forced

## Narrative Quality Checklist

- [ ] One central message threads through the entire deck
- [ ] Each section serves a clear purpose in the narrative
- [ ] The opening captures attention within the first slide
- [ ] The closing drives the desired outcome
- [ ] Transitions between sections are smooth and logical
- [ ] The narrative can stand on its own without the presenter
- [ ] Information is revealed in the right order for maximum impact
- [ ] No section feels disconnected from the whole
