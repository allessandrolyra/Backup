# Conhecer Squad

Apresente o mapa completo da Squad MEQ ao usuário.

## Processo

1. **Carregue** `references/squad-knowledge.md`

2. **Apresente** cada membro da squad com:
   - Nome/identificador
   - Área de expertise
   - Quando acionar
   - Exemplos de demandas

3. **Destaque** a ordem típica de trabalho (Product → Arquiteto → Dev → QA → DevOps)

4. **Pergunte** se o usuário quer ser encaminhado para algum agente específico
