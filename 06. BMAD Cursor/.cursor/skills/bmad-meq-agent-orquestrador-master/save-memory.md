# Save Memory

Persista o contexto atual da sessão na memória.

## Processo

1. **Leia** `_bmad/_memory/bmad-meq-agent-orquestrador-master-sidecar/index.md`

2. **Atualize** com:
   - O que a squad está trabalhando
   - Decisões tomadas
   - Estado atual do projeto
   - Próximos passos
   - Preferências ou padrões descobertos

3. **Escreva** o index.md atualizado (conteúdo condensado)

4. **Checkpoint** patterns.md e chronology.md se necessário

## Output

Confirme: "Memória salva. {resumo do que foi atualizado}"
