# Save Memory

Persista o contexto atual de arquitetura na memória.

## Processo

1. **Leia** `_bmad/_memory/bmad-meq-agent-arquiteto-sidecar/index.md`

2. **Atualize** com:
   - Decisões arquiteturais tomadas
   - Stack escolhido (web, Android, iOS)
   - Padrões e convenções definidos
   - Próximos passos
   - Preferências ou padrões aprendidos

3. **Escreva** o index.md atualizado (conteúdo condensado)

4. **Checkpoint** patterns.md e chronology.md se necessário

## Output

Confirme: "Memória salva. {resumo do que foi atualizado}"
