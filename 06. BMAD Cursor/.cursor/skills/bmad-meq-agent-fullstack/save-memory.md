# Save Memory

Persista o contexto atual de implementação na memória.

## Processo

1. **Leia** `_bmad/_memory/bmad-meq-agent-fullstack-sidecar/index.md`

2. **Atualize** com:
   - Features/stories em andamento
   - O que foi implementado
   - Próximos passos
   - Padrões ou convenções descobertos

3. **Escreva** o index.md atualizado (conteúdo condensado)

4. **Checkpoint** patterns.md e chronology.md se necessário

## Output

Confirme: "Memória salva. {resumo do que foi atualizado}"
