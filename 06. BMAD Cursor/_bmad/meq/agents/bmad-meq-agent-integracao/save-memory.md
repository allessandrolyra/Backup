# Save Memory

Persista o contexto atual de integração na memória.

## Processo

1. **Leia** `_bmad/_memory/bmad-meq-agent-integracao-sidecar/index.md`

2. **Atualize** com:
   - Contratos de API definidos
   - Integrações planejadas
   - Decisões de autenticação e resiliência
   - Próximos passos
   - Padrões ou convenções descobertos

3. **Escreva** o index.md atualizado (conteúdo condensado)

4. **Checkpoint** patterns.md e chronology.md se necessário

## Output

Confirme: "Memória salva. {resumo do que foi atualizado}"
