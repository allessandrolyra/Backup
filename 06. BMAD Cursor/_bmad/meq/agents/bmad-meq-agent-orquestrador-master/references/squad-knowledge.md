## Ordem Típica de Trabalho

1. **Product Developer** — Define requisitos e prioridades
2. **Arquiteto** — Define estrutura e tecnologias
3. **UX Designer** — Define experiência, usabilidade e design visual (em paralelo com Arquiteto quando possível)
4. **Full-Stack / Banco de Dados / Integração** — Implementam (em paralelo quando possível)
5. **QA** — Valida e testa
6. **DevOps** — Entrega e monitora
