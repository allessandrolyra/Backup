# Save Memory

Persista o contexto atual de dados na memória.

## Processo

1. **Leia** `_bmad/_memory/bmad-meq-agent-banco-dados-sidecar/index.md`

2. **Atualize** com:
   - Modelagem e schemas definidos
   - Decisões de índices e otimizações
   - Migrations planejadas
   - Próximos passos
   - Padrões ou convenções descobertos

3. **Escreva** o index.md atualizado (conteúdo condensado)

4. **Checkpoint** patterns.md e chronology.md se necessário

## Output

Confirme: "Memória salva. {resumo do que foi atualizado}"
