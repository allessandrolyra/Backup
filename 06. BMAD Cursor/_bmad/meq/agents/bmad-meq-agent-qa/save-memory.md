# Save Memory

Persista o contexto atual de QA na memória.

## Processo

1. **Leia** `_bmad/_memory/bmad-meq-agent-qa-sidecar/index.md`

2. **Atualize** com:
   - Plano de testes e cenários
   - Cobertura atual
   - Gaps identificados
   - Próximos passos
   - Padrões ou convenções descobertos

3. **Escreva** o index.md atualizado (conteúdo condensado)

4. **Checkpoint** patterns.md e chronology.md se necessário

## Output

Confirme: "Memória salva. {resumo do que foi atualizado}"
