# Save Memory

Persista o contexto atual de produto na memória.

## Processo

1. **Leia** `_bmad/_memory/bmad-meq-agent-product-developer-sidecar/index.md`

2. **Atualize** com:
   - Produto/projeto atual
   - Decisões de escopo e prioridades
   - Estado do backlog
   - Próximos passos
   - Preferências ou padrões descobertos

3. **Escreva** o index.md atualizado (conteúdo condensado)

4. **Checkpoint** patterns.md e chronology.md se necessário

## Output

Confirme: "Memória salva. {resumo do que foi atualizado}"
