Introducao

O Que e o Command Center?

Em vez de um assistente generico para tudo, voce tem acesso a uma equipe inteira de especialistas, cada um com conhecimento profundo na sua area.

Assistente Generico (ChatGPT, Copilot)

Um unico profissional que sabe um pouco de tudo, mas nao e especialista em nada. Voce precisa explicar o contexto toda vez e a resposta pode ser superficial.

Command Center

Uma equipe de consultoria com dezenas de especialistas. Precisa de AWS? Chame o Artemis. UX? Nexus. Sprint? Scrum Master. Cada um ja vem preparado.

Argumentos

Por Que Escolher o Command Center?

Razoes claras para adotar esta plataforma no ambiente corporativo.

Privacidade e LGPD

Os dados nunca saem da sua infraestrutura. Chaves de API ficam em cookies httpOnly seguros. Sem envio para servidores de terceiros.

Custo-Beneficio

Gemini e gratuito. Ollama e 100% local. Sem licenca por usuario. Em 50 pessoas, economize mais de R$50.000/mes vs. ChatGPT/Copilot.

Especializacao Real

Cada agente vem com instrucoes detalhadas (skills) que definem especialidade e forma de trabalho. Sem explicar contexto toda vez.

Autonomia Tecnica

Agentes interagem com o filesystem real: ler, escrever, editar e executar comandos. Nao apenas sugere -- faz.

7 Camadas de Seguranca

Path blocking, read blocking, command classification (30+ padroes), output/write caps, path traversal guards e atomic locks.

Flexibilidade de Modelos

7 provedores. Troque de modelo com um clique. Funciona offline com Ollama.

Escala Simples

Novo especialista? Crie um SKILL.md e o scanner detecta automaticamente. Qualquer pessoa pode criar agentes.

Interface Unificada

Um unico lugar para todos os especialistas, comandos, documentos e acompanhamento. Menos ferramentas, mais produtividade.

Como Comecar a Usar

Em poucos minutos voce tem o sistema rodando.

1

Instalar

Abra o terminal na pasta do projeto e instale as dependencias.

npm install
2

Iniciar

Rode o servidor de desenvolvimento.

npm run dev
3

Configurar

Cole a chave de API do Gemini (gratuito) nas configuracoes.

aistudio.google.com
4

Conversar

Selecione um agente e comece a conversar.

localhost:3000

Requisitos: Node.js v18+ (recomendado v20+).
Para modelos offline, instale o Ollama e rode ollama serve.
