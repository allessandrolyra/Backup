# Público-Alvo: Agente de Onboarding

## Usuários Primários
- **Perfil**: Novos funcionários (todos os níveis).
- **Necessidade**: Orientação rápida, redução de ansiedade, acesso a processos sem interromper colegas.
- **Frustrações**: Excesso de informação, manuais PDF densos, medo de perguntar o "óbvio".

## Usuários Secundários / Stakeholders
- **RH**: Responsável por fornecer a base de conhecimento e validar o aprendizado do agente.
- **Gestores**: Interessados na velocidade de "time-to-productivity" do colaborador.

## Cenários de Uso
1. **Dia 1**: Agente chama o usuário no Teams: "Olá! Sou o seu mentor digital. Para começar, aqui estão os 3 links essenciais para hoje."
2. **Dúvida Pontual**: "Como configuro minha VPN?" (O agente consulta o PDF de TI).
3. **Cultura**: "Qual o dress code da empresa?" (O agente consulta o manual de cultura).
